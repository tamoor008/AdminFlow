# AdminFlow - MOTHERLAND Admin Panel

This is the web-based admin panel for the MOTHERLAND application.

## Getting Started

### Installation

```bash
npm install
```

### Run Development Server

```bash
npm start
```

The app will run on [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
npm run build
```

## Features

- Admin Sign In
- Admin Dashboard
- User Management
- Class Management

## Tech Stack

- React 18
- React Router v6
- Firebase Authentication & Database
